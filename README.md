# Stars & Stripes Canvas

A 4th of July interactive storybook built with HTML & CSS only — no JavaScript.

## Pages

| Page | Description |
|------|-------------|
| Cover | Landing page |
| Freedom's Dawn | History cards |
| Patriot's Plaza | Fireworks over Freedom Harbor |
| United We Stand | Pillars of freedom + pledge quote |

## How it works

- Navigation uses CSS `:target` — no JS needed
- Fireworks button uses a hidden checkbox + `:checked` selector
- All animations use `@keyframes`
- `prefers-reduced-motion` disables animations for accessibility

## Colors
